## 2. Descripción del Servicio

AudioLibro es una herramienta de conversión de texto a audio diseñada para facilitar el acceso a contenidos escritos mediante tecnología de síntesis de voz. La Aplicación permite a los usuarios:

- Subir archivos PDF, TXT y MD
- Convertir texto a audio mediante Web Speech API
- Gestionar una biblioteca personal de audiolibros
- Personalizar la experiencia de lectura para accesibilidad
